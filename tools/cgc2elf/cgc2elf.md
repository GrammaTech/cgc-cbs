# NAME

cgc2elf file [... <file_N>]

# DESCRIPTION

cgc2elf is a utility to convert DECREE format executables to ELF.

## ARGUMENTS

file
: The file to convert

## EXIT STATUS

0
: The files have been converted.

1
: The file is unable to be opened.
